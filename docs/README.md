# Dokumentasjon for ai-guide-book

Prosjektet er et moderne Next.js (React/TypeScript) undervisningsnettsted med ettstående sideinndelinger for KI-tema. Dokumentasjonspakken gir oversikt over arkitektur, mål og utviklingspraksis.

Innhold:
- `struktur.md`: Struktur og filkart for repo
- `utvikling.md`: Lokal utvikling, bygg og kjøring
- `CONTRIBUTING.md`: Bidragsmodell og kvalitetskriterier
